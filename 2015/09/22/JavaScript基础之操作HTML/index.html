4.创建节点

HTML中的所有内容都是节点，节点分为以下几种：

  • 每个HTML元素都是元素节点
  • HTML元素内的文本是文本节点
  • 每个HTML属性是属性节点
  • 注释是注释节点

5.添加节点、替换节点、删除节点

appendChild（从父节点来看）：从该元素子节点的末尾添加子节点
insertBefore（从子节点来看）：从该元素已有的子节点前插入新的子节点，如果是null，则与appendChild相同

replaceChild：将该元素新的子节点替换旧的子节点
removeChild：将该元素的node子节点删除
